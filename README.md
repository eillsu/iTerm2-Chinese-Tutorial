# iTerm2 中文教程说明

![](https://www.iterm2.com/img/logo2x.jpg)

## 前言

iTerm2 是 macOS 平台上最好用的 terminal 程序之一。它的功能十分强大，使用的过程中慢慢了发现很多亮眼的功能，可以极大地提高工作效率。

因时间和精力有限，本教程目前是在逐步完善的过程中。

希望对大家能有一点点帮助。
